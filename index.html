ตั้งค่าระบบ

กำหนดค่าการเชื่อมต่อ ระดับรางวัล และประเภทรางวัล

Cloudinary Configuration

API Key และ API Secret ใช้สำหรับลบไฟล์ออกจาก Cloudinary เมื่อลบผลงานในระบบ — หาได้ที่ Cloudinary Dashboard → Settings → Access Keys
หากไม่กรอก ระบบจะลบเฉพาะข้อมูลในเว็บ แต่ไฟล์ใน Cloudinary จะยังคงอยู่

สร้าง Unsigned preset ที่ Cloudinary Dashboard → Settings → Upload

จัดการปีการศึกษา

เพิ่มหรือลบปีการศึกษาที่แสดงเป็นตัวเลือกในฟอร์มอัพโหลด (พิมพ์ตรงก็ได้เช่นกัน)

จัดการระดับรางวัล

เพิ่ม แก้ไข หรือลบระดับรางวัลที่แสดงในฟอร์มอัพโหลด

จัดการประเภทรางวัล

เพิ่ม แก้ไข หรือลบประเภทรางวัลที่แสดงในฟอร์มอัพโหลด

การตั้งค่าแจ้งเตือนการสมัครสมาชิก

กำหนดอีเมลที่จะรับการแจ้งเตือนเมื่อมีผู้สมัครสมาชิกใหม่ และโหมดการอนุมัติ

โหมดอนุมัติ
เมื่อมีผู้สมัคร ระบบจะรออนุมัติจากผู้ดูแล
อนุมัติเอง อัตโนมัติ
อีเมลที่รับการแจ้งเตือน
การแจ้งเตือนจะถูกบันทึกในระบบ (Notification Log) เนื่องจากนี่คือแอปฝั่ง client-only จึงไม่ส่งอีเมลจริง แต่สามารถเชื่อมต่อ Email API (เช่น EmailJS) ได้ในอนาคต

ประวัติการแจ้งเตือน

บันทึกการแจ้งเตือนที่เกิดขึ้นทั้งหมด (สูงสุด 50 รายการ)
